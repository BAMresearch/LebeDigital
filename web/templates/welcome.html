Automated Workflow

  • Raw Data, like sample matrix and test results, are usually LAB specific and can have different formats: .xls, .csv, .dat… (1)
  • Data and metadata are extracted from the raw files using a Python script and then saved in a standardized JSON file (2, 3)
  • Knowledge Graph (KG) templates following the structure of the Concrete Production and Testing Ontology (CPTO) are designed using a program which generates Draw.io files (4.1, 4.2, 4)
  • Mapping: Placeholders with hashtags # and dollar-symbols $ contained in the KG-templates are replaced by the extracted data and metadata using a Python-script (5)
